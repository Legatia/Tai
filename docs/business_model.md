# Tai Business Model & Phased Rollout

## Core Philosophy
Instead of relying on ads from day one, we bootstrap through **interactive financial primitives** that create value for streamers, viewers, and the platform simultaneously.

**Token Strategy (MVP & Phase 1-2):** Use **SUI** for all economic activities. A **Points System** tracks user contributions for future token airdrop when we launch the native token at the end of Phase 2.

## üéØ Prediction Market Details

### Standard In-Room Prediction
**Use Case:** "Will I get the legendary drop this run?"

**Flow:**
1. Streamer creates prediction (sets question + duration)
2. Viewers bet YES or NO
3. Optimistic Oracle resolves (5-min challenge window)
4. Winners claim share of losing pool

**Economics:**
```
Total Pool: $100
YES Pool: $60
NO Pool: $40
Outcome: YES

Winners (YES bettors) get:
- Principal back: $60
- Share of losing pool: $40 √ó 0.95 = $38
Total: $98 (split pro-rata)

Platform earns: $40 √ó 0.05 = $2
```

---

### Stream Futures (v2)
**Use Case:** "Will streamer X reach 10K followers by Dec 2025?"

**Flow:**
1. Anyone can create a long-term prediction
2. Liquidity accumulates over weeks/months
3. Settlement on target date
4. Platform takes smaller cut (3%) due to higher volume

**Why It Works:**
- Creates "investment thesis" around streamers
- Longer timeframes = more participants
- Natural discovery mechanism (trending futures = trending streamers)

---

### Rage Quit Insurance (v2)
**Use Case:** "Will streamer rage quit within 10 minutes?"

**Flow:**
1. Viewers bet on tilt likelihood
2. Short time window (5-15 min)
3. If rage quit happens, YES wins
4. If not, pool goes to streamer as "tilt tax"

**Why It's Fun:**
- Turns frustration into revenue
- Encourages spicy, high-emotion content
- Fast-paced gambling mechanics

**Economics:**
```
If NO wins (streamer stays calm):
- Streamer gets 95% of YES pool (rewarded for composure)
- Platform gets 5%
- This reduces streamer incentive to fake rage quit
```

## üéÅ Bootstrapping Incentives

### Points-Based Rewards (Instead of Token Airdrops)
Since we don't have a token yet, all early activity earns **Points**:

#### Viewer Quest System
- Watch first stream ‚Üí 100 points
- Tip a streamer ‚Üí 50 points
- Win a prediction ‚Üí 100 points
- Join 5 different rooms ‚Üí 200 points

#### Points Multipliers
- **Streaming:** 10 points/minute
- **Watching:** 1 point/minute
- **Tipping (sent):** 2 points per SUI
- **Tipping (received):** 5 points per SUI
- **Prediction Bets:** 3 points per SUI bet
- **Prediction Wins:** 50 bonus points

### Token Launch (End of Phase 2)
When the native token launches:
```
User's Token Allocation = (User Points / Total Points) √ó Airdrop Pool
```

**Example:**
- Total Points: 10M
- Airdrop Pool: 10M tokens (10% of supply)
- User with 100K points = 1% of airdrop = 100K tokens

### Zero Platform Fees (Phase 1)
- 0% on tips during first 3 months
- Builds loyalty before introducing fees
